Boxplots in Pandas

Thu 16 January 2014

Objective: A single plot with box plots of multiple data sets side by side.

I recently wanted to plot each of several groups of data (stored in a single column in my data frame) as a single plot with the boxplots corresponding to each group of data lined up side by side.

The functionality that Pandas provides for this comes from the call pandas.DataFrame.boxplot(). I could not find this functionality out-of-the-box with Pandas. I was able to produce the desired plot by creating a pivot table with my data, but I believe the approach here is much simpler.
